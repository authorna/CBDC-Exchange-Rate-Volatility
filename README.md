# CBDC-Exchange-Rate-Volatility

This repository contains the data and R replication code for the manuscript:

**“Do Central Bank Digital Currencies Affect Exchange Rate Volatility?”**

## Description

This study examines the impact of Central Bank Digital Currency (CBDC) adoption on Real Effective Exchange Rate (REER) volatility across seven early adopters: Nigeria, Kazakhstan, India, Jamaica, the Bahamas, Brazil, and China over the period 2016–2024.

The empirical strategy combines:

* GJR-GARCH volatility modeling
* OLS regressions
* Fixed-effects panel estimations

## Repository Structure

### `data/`

Contains the datasets used in the empirical analysis.

### `rcode/`

Contains the R scripts used to generate all tables of the manuscript.
